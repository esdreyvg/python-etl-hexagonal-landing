// FAQ

Preguntas frecuentes

¿Funciona con Python 3.10?
El kit está diseñado para Python 3.11+ por el uso de type hints modernos (X | Y en vez de Optional[X]). Con pequeños ajustes en las anotaciones funciona en 3.10.
¿Puedo agregar un loader a base de datos distinta de Parquet?
Sí. La guía docs/adding-new-source.md muestra exactamente cómo extender el sistema sin modificar código existente. El mismo patrón aplica para loaders: creas un archivo nuevo y lo registras en PipelineBuilder.
¿Los tests corren sin base de datos?
Los 20+ tests unitarios usan doubles (FakeExtractor, FakeLoader) y corren en menos de 1 segundo sin ninguna dependencia externa. El test de integración sí requiere las librerías instaladas pero no necesita servidor.
¿Puedo usarlo en proyectos comerciales?
Sí, la licencia permite uso personal y comercial. No está permitida la redistribución del template como producto.
¿El extractor REST funciona con autenticación Bearer y API Keys?
Sí. Los headers se configuran en el JSON (incluyendo Authorization) y las credenciales se resuelven desde variables de entorno con la sintaxis ${VARIABLE}, nunca hardcodeadas.
